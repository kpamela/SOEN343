## Web stack
**View** | ReactJS
**Style** | SCSS
**HTTP** | Axios
**Webserver** | NodeJS/Express
**Database** | MySQL
**Bundler** | Webpack

## Package install
```bash
npm install --save
```
***Install XAMPP before running the package

## Start the local server
```bash
npm start
```
Then go to http://localhost:3000/

## Run the Unit tests
```bash
npm test
```
