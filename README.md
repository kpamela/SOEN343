## Web stack
**View** | ReactJS
**Style** | SCSS
**HTTP** | Axios
**Webserver** | NodeJS/Express
**Database** | MySQL
**Bundler** | Webpack

## Package install
```bash
npm install --save
```

## Start the local server
```bash
npm start
```
Then go to http://localhost:3000/
